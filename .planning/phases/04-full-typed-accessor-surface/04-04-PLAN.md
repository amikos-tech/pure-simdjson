<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| public iterator API -> hidden iterator transport | Public methods must not leak raw iterator-state assumptions or continue after doc closure |
| direct field helpers -> scalar string access | The helper must preserve the same missing/null/wrong-type semantics as the underlying primitives |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04-01 | T | iterator closed-state | mitigate | Tie iterators to the owning doc, stop iteration after closure, and surface terminal failure via `Err()` |
| T-04-04-02 | T | missing-vs-null lookup | mitigate | Keep `Object.GetField(...)` on the native `ERR_ELEMENT_NOT_FOUND` path and let null remain a valid returned `Element` |
| T-04-04-03 | I | `GetStringField(name)` helper | mitigate | Implement the helper as explicit `GetField(name)` + `GetString()` composition so its behavior cannot drift from the primitives |
</threat_model>
