**Fault injection tests (require httptest + injected httpClient):**
Note: all these tests inject the httptest server URL via `WithMirror(srv.URL)` and pass `withHTTPClient(srv.Client())`. They call `BootstrapSync(ctx, WithDest(t.TempDir()), WithMirror(srvURL), withHTTPClient(client), WithTarget("linux", "amd64"))`. For GitHub fallback tests, use a two-server setup (R2 mock + GH mock) with a test option `withGitHubBaseURL`.

- `TestBootstrapSync` — httptest server returns fakeLibBody; Checksums temporarily patched to match; BootstrapSync succeeds; artifact exists at expected path
- `TestBootstrapSyncCancellation` — server sleeps 200ms per read chunk; ctx cancelled after first 1KB; BootstrapSync returns context.Canceled or context.DeadlineExceeded; temp file not left in cacheDir
- `TestBootstrapSyncCtxCancelDuringSleep` — server returns HTTP 429 on first attempt; ctx cancelled before retry sleep expires; BootstrapSync returns within 50ms; measured via time.Since
- `TestCorruptedDownloadRejected` — server returns bytes with flipped bit; Checksums set to expected good hash; BootstrapSync returns error wrapping ErrChecksumMismatch; no artifact file written
- `TestRetryOn429ThenSuccess` — server returns 429 first attempt then 200; BootstrapSync succeeds; hit counter is 2
- `TestFallback503R2Then200GH` — R2 mock returns 503 on all 3 attempts; GH mock returns 200; BootstrapSync succeeds using GH fallback
- `TestFallback404R2Then200GH` — R2 mock returns 404; GH mock returns 200; BootstrapSync succeeds using GH fallback
- `TestDisableGHFallbackWith404` — PURE_SIMDJSON_DISABLE_GH_FALLBACK=1; R2 mock returns 404; BootstrapSync returns error wrapping ErrAllSourcesFailed; GH mock gets 0 hits
- `TestMirrorOverride` — PURE_SIMDJSON_BINARY_MIRROR=customServer.URL; download hits custom server, not defaultR2BaseURL
- `TestRedirectDowngradeRejected` — server returns 301 redirect to an http:// URL; BootstrapSync returns permanent error; hits counter is 1 (no retry)
- `TestGitHub403RateLimit` — server returns 403 with body `{"message": "API rate limit exceeded..."}` twice then 200; BootstrapSync succeeds (403 retried as rate-limit)
- `TestConcurrentBootstrap` — 8 goroutines call BootstrapSync simultaneously; WaitGroup waits for all; no errors; hit counter == 1 (flock ensures single download)
