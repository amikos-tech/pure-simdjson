| Boundary | Description |
|----------|-------------|
| public ABI handles -> internal runtime registry | Incorrect packing or generation checks would become stale-handle corruption or accidental aliasing |
| parser lifecycle -> bridge/native document | Hidden parser reuse would invalidate documents silently instead of failing loudly |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-02-02-02 | T | parser lifecycle | mitigate | Enforce one live doc per parser and return `PURE_SIMDJSON_ERR_PARSER_BUSY` until `doc_free` succeeds |
