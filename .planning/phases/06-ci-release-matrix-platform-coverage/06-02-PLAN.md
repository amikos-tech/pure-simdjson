- `v<version>/linux-amd64/libpure_simdjson.so`
- `v<version>/linux-arm64/libpure_simdjson.so`
- GitHub copies `libpure_simdjson-linux-amd64.so` and `libpure_simdjson-linux-arm64.so`

Mirror that same check inside `scripts/release/build_linux_manylinux.sh` or a helper it calls so the proof remains part of the versioned linux build path rather than a one-off inline YAML fragment. Upload `linux-arm64-pagesize.txt` from the prep workflow alongside the packaged arm64 artifact so PLAT-02 retains a concrete evidence file.

- `verify_reproducible: true` in `release-prepare.yml`

- `cargo build --release --locked` through the shared action from Plan 06-01

When wiring artifact transfer for later staged-tree assembly, add `actions/upload-artifact` / `actions/download-artifact` using full commit SHAs, not `@v4` or other moving tags.
